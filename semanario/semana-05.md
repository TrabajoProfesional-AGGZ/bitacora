# Semana 5: Refinamiento de Ideas y Análisis Táctico

**Período:** Jueves 9 de Abril al Miércoles 15 de Abril.

## 🎯 Objetivos de la semana

* [x] Participar en la segunda reunión cíclica con el tutor.
* [x] Replantear y filtrar las propuestas principales según el feedback recibido.
* [x] Seleccionar las 3 mejores ideas para la presentación del jueves.
* [x] Realizar el análisis de las "4 P" las 3 mejores ideas. Y completar las "5 C" para las nuevas propuestas líderes.
* [x] Actualizar la documentación técnica en el repositorio de Análisis de Productos.

## 📝 Tareas realizadas

* **Segunda Reunión Cíclica:** Participación en la sesión de tutoría donde se recibió feedback crítico sobre la viabilidad y el enfoque de las 4 ideas iniciales.
* **Filtrado y Selección de Propuestas:** Basándose en las correcciones del tutor, el equipo realizó un proceso de descarte, pivotaje y creación de nuevas propuestas. Se seleccionaron las 3 ideas con mayor potencial técnico y comercial para la presentación del próximo encuentro.
* **Profundización del Análisis (4P y 5C):** Se elaboró el análisis táctico de las "4 P" para la terna finalista. Además, para las 2 ideas que surgieron como nuevas favoritas, se completó el análisis de las "5 C".
* **Consolidación Documental:** Carga de los nuevos análisis, fórmulas de proyección de ingresos y reestructuración de los documentos `.md` en el repositorio de trabajo.

## 🚧 Problemas / Desafíos

* **Reestructuración de Propuestas:** El principal desafío fue alinear la visión del equipo con los requisitos académicos, técnicos y comerciales indicados por el tutor. Varias ideas que parecían sólidas fueron descartadas o debieron cambiar radicalmente su enfoque para ser mejor recibidas.
* **Resolución:** Se realizaron múltiples reuniones semanales de equipo para replantear los conceptos. Se utilizaron componentes de ideas descartadas para fortalecer las propuestas actuales e incluso se gestó una nueva propuesta integradora que cumple mejor con los objetivos del Trabajo Profesional.

## ⏭️ Próximos pasos

* **Tercera Reunión Cíclica:** Presentación de las 3 ideas finales pulidas y mejoradas. El objetivo es obtener el visto bueno definitivo del tutor para la elección del producto final.
* **Selección de Idea Final:** En base al nuevo feedback, decidir por cuál de las opciones optar y comenzar el desarrollo de los requerimientos específicos.
* **Inicio de Fase Operativa:** Una vez validada la idea mejor posicionada, comenzar con el trabajo detallado sobre esa propuesta única.
