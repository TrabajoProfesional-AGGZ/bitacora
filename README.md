# Bitácora de Avances

Registro semanal, cronograma de hitos y seguimiento continuo del progreso del Trabajo Profesional.

📖 **[Ver la documentación online aquí](https://trabajoprofesional-aggz.github.io/bitacora/)**

## Sobre este repositorio

Este espacio documenta el control de avances, las tareas realizadas y el estado actual del proyecto semana a semana.

Cada entrada de la bitácora detalla los objetivos planificados, las actividades completadas, los problemas o bloqueos encontrados y los próximos pasos a seguir, asegurando un historial claro de la evolución hasta el producto final.

Desarrollado con [Just the Docs](https://just-the-docs.com/).
